## Usage

### docker

use docker to deploy

### Use the source code

环境要求：请先配置好jdk环境以及maven环境

使用方法：将整个项目包下载到本地，再pom.xml目录进入cmd界面，输入mvn spring-boot:run即可启动项目
         在浏览器本地8080端口即可访问

注意：请在resources目录下的application.properties修改数据库配置，换成自己数据库配置即可
